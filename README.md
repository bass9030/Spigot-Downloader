Spigot Downloader
======
Easy download Minecraft Spigot files!<br>
마인크래프트 스피곳 파일을 쉽게 다운로드할 수 있습니다!

## Korean(한국어)
그동안 스피곳 파일을 다운받기 위해 git을 설치하고 명령어를 치는 번거로움을 한번에 날려드립니다.<br>
일부 메뉴만 설정해주면 쉽게 스피곳을 다운로드 할수 있습니다.

**해당 프로그램은 .Net framework 4를 필요로 합니다.**
